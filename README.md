# Weather App

## Overview

This project is a responsive Weather Application built using HTML, CSS, and JavaScript. It fetches real-time weather information from the OpenWeatherMap REST API and displays weather details for any city entered by the user.

---

## Features

- Search weather by city name
- Real-time weather updates
- Loading indicator while fetching data
- Error handling for invalid inputs
- Responsive design
- Dynamic DOM manipulation using JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- OpenWeatherMap API

---

## API Information

API Provider:
https://openweathermap.org/api

Endpoint Used:

https://api.openweathermap.org/data/2.5/weather

Data Retrieved:
- City Name
- Temperature
- Weather Description
- Humidity
- Wind Speed
